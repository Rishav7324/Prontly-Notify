USAGE RULES:
- Always use rgba backgrounds, never solid fills on glass components
- backdrop-filter requires the parent element to have a defined background (not transparent)
- Use sparingly — too many blurred layers stack and degrade performance. Max 3 simultaneous blur layers.
- On mobile: reduce blur from 20px→12px for performance
- Fallback (when backdrop-filter unsupported): use rgba(17,24,39,0.95) — more opaque solid version
- Never combine a heavy drop-shadow AND a large blur on the same element — pick one primary depth capture
